Software Project Registration Form



\## Description

This project is a Java Swing desktop application that allows users to register software projects.



The application collects project information from a form and saves the data to a text file using File I/O.



\##############################################################################################################

\##############################################################################################################



How to Run

1\. Open the project in an IDE.

2\. Run `ProjectFormApp.java`.

3\. Fill in the form and click \*\*Save\*\* to store project data.
